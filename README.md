# To Do List — Vol 2.0.1 (2026)

A clean, UI-focused **To Do List** built with vanilla HTML, CSS, and JavaScript.  
Add tasks, edit them, delete them, and mark them complete — with a live “pending tasks” counter and simple input validation.

---

## Features

- **Add tasks** instantly from the input field
- **Edit tasks** (pulls the task text back into the input for quick changes)
- **Delete tasks**
- **Mark as complete** with a custom checkbox + strike-through styling
- **Live pending counter** updates as you add/complete/delete
- **Empty input handling** with a clear error message

---

## Built With

- **HTML5**
- **CSS3** (CSS variables, grid layout, custom checkbox styling)
- **JavaScript (ES6)** (DOM manipulation + event handling)
- **Font Awesome** icons + **Google Fonts (Poppins)**
